# Branding y Paleta de Colores

- Negro: #000000
- Celeste: #0ec0f2 (alternativo: #22c1e7)
- Rojo: #e60023 (alternativo: #d7263d)
- Blanco: #ffffff

Guía de uso
- Fondo principal: negro o muy oscuro para un look premium.
- CTAs: celeste para acciones primarias; rojo para urgencias o acentos fuertes.
- Texto: blanco o gris claro para legibilidad; mantener contraste alto con fondos oscuros.
- Acabados: usa iconografía minimalista en blanco, con toques de rojo/celeste en acentos.

Notas
- Mantén consistencia en la aplicación de la paleta a través de todas las secciones.
- Evita saturar con colores diferentes; prioriza jerarquía visual por tamaño y contraste.
